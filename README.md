Desarrollo de una landing page responsive utilizando Bootstrap, aplicando principios de diseño mobile-first y buenas prácticas de maquetación web. El proyecto incluye estructura semántica en HTML, uso de componentes reutilizables de Bootstrap y adaptación a múltiples tamaños de pantalla.

Este trabajo me permitió consolidar conocimientos en diseño responsive, organización de layouts y uso eficiente de frameworks CSS.
